# AlphaBeta
## 介绍
这是一个浏览器，不同于其它浏览器，不基于任何现有的浏览器内核，仅使用了bs4解析html格式，tkinter渲染画面，requests获取html内容。
## 使用的东西
Requests - 获取HTML内容\
Python - 编写程序\
Tkinter - 渲染画面\
BeautifulSoup - 解析HTML格式
## 安装教程
Windows运行install.bat，Linux/MacOS运行install.sh（必须配置好环境变量！但是如果是release编译好的版本则不用，因为你也不用）\
然后确保安装完成后，运行main.py
